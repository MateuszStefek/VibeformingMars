# Project: GHG Producing Bacteria (#034)

## Card Data
- **Included in:** Base Game (Corporate Era)
- **Cost:** 8 MC
- **Tags:** Science, Microbe
- **Requirements:** 4% Oxygen
- **Action:** Either add 1 Microbe to this card, or remove 2 Microbes from this card to raise the temperature 1 step.


## Strategic Analysis (The Microbe-Heat Engine)
**Status: B-TIER UTILITY / S-TIER WITH EXTREME-COLD FUNGUS**

GHG Producing Bacteria is a high-efficiency terraforming engine that converts time (microbe accumulation) into global parameter progress. While slow on its own, it becomes a top-tier point generator when paired with microbe-boosting cards.

### Why it is a top-tier play:
1. **Low-Cost TR:** For only 8 MC (+3 to draft), it provides a repeatable way to raise the temperature. In a typical game, playing it early usually yields 2-3 TR, representing a massive discount over the 14 MC Standard Project temperature raise.
2. **Double Tag Synergy:** Provides both a **Science** and a **Microbe** tag.
   - **Science:** Critical for hitting requirements for *AI Central* or *Antigravity Technology*.
   - **Microbe:** Triggers *Viral Enhancers*, *Decomposers*, and *Insects*.
3. **Synergies:**
   - **Extreme-Cold Fungus (ECF):** The "God Tier" combo. ECF adds 2 microbes per turn, allowing you to **raise the temperature every single generation**.
   - **Enceladus (Colonies):** Allows you to trade for microbes to bypass the slow buildup, potentially terraforming twice as fast.
   - **Splice Tactical Genomics:** Triggers Splice's ability for immediate resource or card advantage.

### The "Trap" Check:
- **Requirement Barrier:** The 4% Oxygen requirement means you cannot play this in the very early game. Do not hold it in your opening hand unless you have a path to rapid Oxygen growth or the **Inventrix** corporation.
- **Slow Standalone Pace:** Without boosters, it takes 3 generations for a single TR raise. In a fast-paced "Rush" game, it may never trigger more than once.
- **Ants Predator:** **Ants** can steal the microbes from GHG Bacteria. If an opponent has Ants, prioritize raising the temperature immediately rather than hoarding microbes.

### Tips
- **Requirement Buffer:** Use GHG Bacteria primarily as a cheap Science tag to unlock *AI Central*. The terraforming action is a bonus "side-engine" that provides steady value once the core engine is running.

**Recommendation:** A high-priority draft if you have any microbe-boosting cards or need a cheap Science tag. Priority 1 if holding *Extreme-Cold Fungus*. Skip late if temperature is already nearing the max.
