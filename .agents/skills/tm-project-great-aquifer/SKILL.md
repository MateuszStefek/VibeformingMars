# Project: Great Aquifer (#114)

## Card Data
- **Included in:** Base Game (Basic)
- **Cost:** 17 MC
- **Tags:** None (Event)
- **Requirements:** None
- **Effect:** Place 2 Ocean tiles.


## Strategic Analysis (The Ocean Surge)
**Status: A-TIER EFFICIENCY / PREMIER RUSH EVENT**

Great Aquifer is a high-efficiency terraforming card that provides massive board impact and TR gain for a relatively low cost. It is one of the most reliable tools for accelerating the game's end while securing high-value board positions.

### Why it is a top-tier play:
1. **Exceptional Cost Efficiency:** At 17 MC (+3 to draft), you are paying **10 MC per ocean**. This is a significant discount over the 18 MC Standard Project ocean. It is the cheapest way to place two oceans in a single action without high requirements.
2. **Immediate TR & Board Control:** Provides 2 TR instantly, increasing your income and score. Placing two oceans simultaneously allows you to claim high-value placement bonuses (e.g., 2 titanium or 2 cards) and "walk" your way to strategic board positions.
3. **Synergies:**
   - **Arctic Algae:** The premier synergy. Gain 4 plants immediately upon placement, often leading to a greenery conversion in the same turn.
   - **Media Group:** As an Event card, it triggers the 3 MC rebate, making the 2 oceans cost only 14 MC total—an unbeatable ROI.
   - **Lakefront Resorts:** Provides an immediate income boost upon placement.
   - **UNMI:** Provides the 2 TR needed to trigger UNMI's corporation ability in Generation 1-2.

### The "Trap" Check:
- **Zero Tags:** The card provides zero progress toward milestones like *Builder* or *Ecologist*. If you are racing for tag-based awards, the 20 MC might be better spent elsewhere.
- **Ocean Cap:** Monitor the ocean track. In high-player games, oceans are often the first parameter to finish. Do not draft this late if only 1 ocean spot remains.

### Tips
- **Bonus Sniping:** Use the double placement to grab the "2 card" and "2 titanium" spots in a single move before opponents can react.
- **Requirement Bumping:** Use Great Aquifer to quickly reach the 6-ocean requirement for **Kelp Farming** or the 5-ocean requirement for **Algae**.

**Recommendation:** A high-priority draft for any strategy. High priority for **Rush** players or anyone holding *Arctic Algae*. Skip only if your engine is already over-optimized and you need card draw instead of TR.
