# Project: Comet (#010)

## Card Data
- **Included in:** Base Game (Basic)
- **Cost:** 21 MC
- **Tags:** Space, Event
- **Requirements:** None
- **Effect:** Raise temperature 1 step. Place 1 ocean tile. Remove up to 3 plants from any player.


## Strategic Analysis (The Foundation Terraformer)
**Status: B-TIER / EFFICIENT RUSH EVENT**

Comet is a staple terraforming card that provides high efficiency and board presence. It is a key tool for players looking to accelerate the game's end while gaining a foothold on the board.

### Why it is a top-tier play:
1. **Cost-to-TR Efficiency:** For a total cost of 24 MC (21 + 3 to draft), you gain 2 TR (1 from temperature, 1 from ocean) and a placement bonus. This is slightly better than standard project costs, especially when accounting for the ocean bonus.
2. **Space Tag Utility:** As a Space card, it can be paid for with Titanium. This makes it an excellent "budget" way to place an ocean compared to the 36 MC *Giant Ice Asteroid*.
3. **Tactical Disruption:** Removing 3 plants can be a game-saver if it prevents an opponent from reaching the 8-plant threshold for a greenery.
4. **Synergies:**
   - **Arctic Algae:** Gain 2 plants immediately upon placing the ocean.
   - **Optimal Aerobraking:** Provides 3 MC and 3 Heat, making the effective cost of the TR jump and ocean very low.
   - **Media Group:** Provides a 3 MC rebate.
   - **Ocean Placement Bonuses:** Allows you to grab a 2-titanium or 2-card spot on the map early.

### The "Trap" Check:
- **Low Impact:** While efficient, it only moves the global parameters by 1 step each. In the late game, it is a point-filler, not a game-changer.
- **Timing:** Most effective in Generations 1-4 when placement bonuses are still available and plant denial has the most impact.

### Tips
- **Ocean Walking:** Use the comet to place an ocean near a lucrative greenery spot you plan to take later. 
- **The "3 Plant" Snipe:** If an opponent has 5-7 plants, this is your primary tool to knock them back and buy yourself a generation of time.

**Recommendation:** A solid "Always-Draft" for terraforming-heavy strategies. High priority if you have titanium production or *Optimal Aerobraking* in play. Use it early to secure board bonuses.
