# Corporation: Kuiper Cooperative

## Corp Data

- **Included in:** Promo
- **Starting MC:** 33 MC, 1 Titanium Prod
- **Ability:** Asteroid resource engine for space-focused terraforming (Oceans/Temp).

**Status: B-TIER / SPACE**

Excellent at pushing global parameters via space events and asteroid projects.

## Map Analysis

Prime map for 'Rim Settler' milestone. The ocean placement bonuses are great for titanium.
### Elysium
Target the 'Celebrity' award. High-value cards are your primary scoring engine.
